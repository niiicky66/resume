dria.co 字体风格 · 我们在借鉴的对象

无法从外部直接读到 dria.co 的 CSS 字体声明，以下是基于其视觉风格的最接近匹配（建议你在站点上右键 Inspect → font-family 实测核实）：dria 走的是当下 AI 品牌的暖调高对比文学衬线大标题 + 中性无衬线正文。本 lab 中最贴近的展示档是 Newsreader 或 EB Garamond（古典优雅档），正文配 Inter / Geist。已在选项里用 ◎ 标出"dria 风格最近似"。

更正 · Source Serif 4 的默认数字

它默认其实是等高比例数字 (Lining)，不是旧体——旧体 3/4/5/7/9 下沉那个效果要手动开 oldstyle-nums。你喜欢的"4 沉到基线下"飘逸风，已做成数字层的「旧体」开关；下方样机里 0–9 对比块实时演示，下沉位高亮成火焰色。

关于中文字体：PingFang SC（苹方）、Songti SC（宋体）、Kaiti（楷体）只存在于 iOS / macOS，无法网络加载；Web / 安卓会自动降级到 Noto Sans/Serif SC（可联网加载）。霞鹜文楷 (LXGW WenKai TC) 我直接联网加载了，所以全平台可见。OPPO Sans / HarmonyOS Sans / 汉仪旗黑 体量大且需授权，未联网加载——本机没装就会显示"缺失"，这正是渲染检测要告诉你的。

我的最终推荐（已用 ● 标在选项上）

展示Newsreader + 思源宋体 Noto Serif SC — 暖调高对比文学衬线，最贴 dria.co；想更古典可换 EB Garamond。
UIInter + 苹方 PingFang SC（Web 落 Noto Sans）— 复用你 RSS 栈，界面最干净。
正文Source Serif 4 + Noto Serif SC — 衬线治"4 年消金运营"那段的寡淡，更有书卷重量。
数字继承正文 + 旧体 Oldstyle · 比例 — 就是你爱的 3/4/5/7/9 下沉飘逸；做仪表盘要列对齐时再切「等宽 Tabular」。
